# Title Case

Title Case converts text between common letter-case formats and Caesar
cipher encode/decode modes.

## Features

- Convert text as you type (title, sentence, upper, lower).
- Caesar encode and decode modes in the same selector.
- Custom Caesar shift input that appears only in Caesar modes.
- Shared Caesar utility implementation with secret-santa.

## Run This App

From repo root:

```bash
yarn start:title-case
```

Build and test from repo root:

```bash
yarn build:title-case
yarn test:title-case
```
