# Secret Santa

Secret Santa is a small app for creating gift exchange assignments and sharing
them as plain names or encrypted tokens.

## Features

- Add participants and generate shuffled assignments.
- Optionally encrypt assignment tokens for safer sharing.
- Decrypt an assignment token manually or from a route parameter.
- Hide/reveal all generated assignments in the results panel.

## Run This App

From repo root:

```bash
yarn start:secret-santa
```

Build and test from repo root:

```bash
yarn build:secret-santa
yarn test:secret-santa
```
