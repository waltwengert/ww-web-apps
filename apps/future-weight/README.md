# Future Weight

Future Weight is a quick calculator for metabolic metrics based on user
stats and activity level.

## Features

- Calculates BMR, TDEE, and BMI from age, height, and weight.
- Supports metric and imperial units.
- Supports male/female formula toggle.
- Auto-updates results after first successful calculation.

## Run This App

From repo root:

```bash
yarn start:future-weight
```

Build and test from repo root:

```bash
yarn build:future-weight
yarn test:future-weight
```
