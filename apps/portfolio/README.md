# Portfolio

Portfolio is a single-page personal site with section navigation for about,
projects, education, and employment.

## Features

- Sticky section navigation that tracks the currently visible section.
- Project tiles with links to source code and live demos.
- Responsive layout tuned for desktop and mobile.
- Basic accessibility improvements (labeled icon links, heading order fixes).

## Run This App

From repo root:

```bash
yarn start:portfolio
```

Build and test from repo root:

```bash
yarn build:portfolio
yarn test:portfolio
```
